# Notices

TinyPyk is an independent open-source project for child-friendly Python
learning.

## Blockly

TinyPyk uses Blockly and currently depends on `blockly@13.0.0-beta.8` from the
Raspberry Pi Foundation Blockly release line. Thank you to the Blockly team and
the Raspberry Pi Foundation for the tools that make block-based learning
possible.

## Learning Inspiration

TinyPyk uses a Scratch-style creative coding workspace with characters, stage,
costumes, sounds, and visible output. TinyPyk is also inspired by Hedy's
progressive approach to introducing Python ideas level by level.

These references describe learning and interface inspiration only. TinyPyk is
not affiliated with Scratch, Hedy, Python, Blockly, or the Raspberry Pi
Foundation unless a separate written agreement says otherwise.

## Third-Party Services

OpenAI and ElevenLabs integrations are optional and disabled unless configured
by an adult administrator on the server. API keys must not be committed to this
repository.
